# 执行计划（短期，每次执行读取）

本计划文件给出下一个开发周期的具体任务列表，遵循“先计划再执行”。建议每次执行前更新本文件，再据此落地实现。

## 当前周期目标（建议）
1. 旗杆滑落与结算演出（计分动画、剩时逐步换分）。
2. BGM 与计时警报：基础循环、时间<100 时节奏加快。
3. 角色动画细化（起跑/急停/空中/落地）、壳滑粒子强化。
4. 关卡 1-4（城堡雏形）：火条 + 桥终点演出（暂不做 Boss）。
5. 数据组织：关卡时间与计分配置拆出到 JSON（仅 1-1/1-2/1-3/1-4）。

## 验收标准
- 滑旗时有演出动画，最终得分包含旗杆高度与剩时换分；
- BGM 正常循环；时间警报触发节奏变化；
- 新增动画与粒子在 60fps 下稳定；
- 1-4 可通关；
- 新 JSON 配置能控制每关时间/旗杆分倍率。

## 备注
- 仍保持零外部依赖；音频以 WebAudio 合成为主；
- 若遇到移动端性能问题，优先降级粒子数量与渲染分辨率。
